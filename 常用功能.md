

## TC操作

| 快捷键       | 功能说明                               | 备注     |
| :----------- | :------------------------------------- | :------- |
| **F6**       | **移动**选定的文件/文件夹到另一窗口    | 核心功能 |
| **F5**       | **复制**选定的文件/文件夹到另一窗口    | 核心功能 |
| **Shift+F5** | 在同一文件夹内**复制**文件（需重命名） |          |
| **Shift+F6** | 在同一文件夹内**重命名**文件           |          |
| **Ctrl+M**   | **批量重命名**文件                     | 王牌功能 |



- **快速选择与导航**
    - **选择文件**：除了鼠标，可以多用**空格键**逐个选择/取消选择文件，这在处理分散文件时非常高效。
    - **切换窗口**：使用 **`Tab`** 键可以在左右两个窗口之间快速切换焦点。
    - **快速定位**：在文件列表里直接键入文件或文件夹的名称，TC会自动定位，这在长文件列表中查找时非常迅速。

- **关于重命名快捷键的说明**
    你可能注意到，在标准的Windows资源管理器中，"重命名"的快捷键是F2。但在TC的**默认设置**中，**`F2` 键的功能是刷新文件列表**，而重命名和移动共用了 **`F6`** 键。
    不过，由于用户习惯的力量，很多人都会将TC的"重命名"功能自定义为F2，这可以通过修改TC的配置实现。你可以通过 **`配置`** -> **`设置...`** -> **`操作`** 来自定义这些快捷键。

- **进阶移动/复制技巧**
    - 如果想移动或复制**当前文件夹下的所有内容（包括子文件夹中的文件）**，可以先用 **`Ctrl+B`** 快捷键，它会将当前目录及其所有子目录下的文件平铺展示在一个虚拟列表中。然后你可以全选（`Ctrl+A`），再使用`F5`或`F6`进行操作。
    - 使用 **`Alt+F7`** 调出强大的搜索功能，找到分布在各个角落的文件后，可以直接在搜索结果中进行移动或复制操作。
